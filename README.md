# Proyecto Innotech

Este Proyecto tiene como meta ser el primer sistema de administración, control y desarrollo del Club Innotech. La meta principal del Club es ayudar a los compañeros y a la carrera con actividades y compartiendo un poco de nuestra ayuda como el unico club hasta ahora de la carrera de  __Tecnologías de la Información e innovación Digital__.

## Principales módulos de la aplicación
